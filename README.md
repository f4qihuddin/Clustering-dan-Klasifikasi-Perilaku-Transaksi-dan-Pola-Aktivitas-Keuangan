# Clustering-dan-Klasifikasi-Perilaku-Transaksi-dan-Pola-Aktivitas-Keuangan
Membuat model clustering dan klasifikasi untuk mengklasifikasikan pola keuangan. Clustering digunakan untuk mengelompokkan pembeli berdasarkan pola-pola pembelian dan aktivitas keuangannya dan kemudian dilakukan klasifikasi
